# Virtual World Logic Creation Protocol (VLCP)

本协议定义了一套创造逻辑严谨、规则合理的虚拟世界的评定标准和价值体系，及遵守该协议后的创作者行为守则，其目的在于：

让创作者创造的虚拟世界的设定和展示过程变得逻辑严谨、规则合理。

协议具体内容可见[VLCP.md](./VLCP.md)。历史版本可预见[VLCPHistory](./VLCPHistory)。

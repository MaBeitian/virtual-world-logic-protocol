# 虚拟世界逻辑性协议

# Virtual World Logic Protocol (VWLP)

美有千万种，语言有千万个，但人类有一样最一致的东西，就是逻辑。

本协议定义了一套创造逻辑严谨、规则合理的虚拟世界的评定标准和价值体系，及遵守该协议后的创作者行为守则，其目的在于：
让创作者创造的虚拟世界的设定和展示过程变得逻辑严谨、规则合理。

本协议的核心文件是一张规则链表。您需要做的最核心的工作就是，为您的虚拟世界，遵循协议内的指引，打造这样的一张规则链表，然后基于这张链表推演您的世界。是的，是“推演”而不是单纯的“演绎”。

本协议的理念是通过设定逻辑严密的规则链及元设定，推演出一个虚拟世界。我们对于创造虚拟世界的理念，不像是一位雕刻家——一个边一个角地雕琢出一个世界，反而更像是农民，我们埋下一颗元设定的种子，浇上逻辑与规则的水，让它长成一个有过去、有现在、有未来的虚拟世界。
如果在过去，这样的理念是难以落地的。因为规则被设定后，进行逻辑局部自洽的、较为严谨的推演，所需的工作量，会超出创作者的承受限度。但是，有了ai agent teams这类人工智能蜂群，这个理念便有机会跨越重重虚拟世界，来到现实。

而在本协议的指导下，创造虚拟世界的理想过程过程是：设定初始条件，然后利用AI开始推演，进而得到广度和深度远超人类作者所能及的广阔虚拟世界。

当然，本协议的原则是，不去干涉虚拟世界创造者的创作自由，在虚拟世界的创作方法和理念上，不拉踩。

本协议的确要求遵守者建立若干个包含特定内容的文档，但是除此之外，不会去干涉创作者自己的已有的目录系统。

本协议不认为自身就是绝对正确的，更不认为所有虚拟世界的创作都要遵照本协议。这个协议是存在一个立场的，它很像是一个虚拟世界创作流派的代表，该流派崇尚通过逻辑推演创造虚拟世界和故事，并尊崇逻辑性强的虚拟世界。
所以，有了一个立场，就意味着存在对立的立场，以及更多共存的立场。流派也是一样的道理。但也正因百花齐放，多元共存，立场才是立场，流派才是流派，才说“本协议存在一个立场”。

本协议不去争辩人工智能创作和人类创作孰优孰劣，也不去讨论该不该使用人工智能创作这类话题。本协议本就是一个开放的、来源的、无强制性的、不自大的、包容性强的、符合逻辑的协议。该协议可以与人工智能很好地协作，也可以用于纯人工创作，更可以用于人机混合的创作。

本协议的愿景是创作者们可以更好地创作虚拟世界，让粉丝们可以通过逻辑推演，更热烈地讨论虚拟世界发生的一切。所以我们欢迎您为这个协议体系提出建议甚至建设。

# 一、前言

本“角色元设定”表模板为《虚拟世界逻辑性协议》（*Virtual World Logic Protocol,VWLP*）的附属文档，其所使用的名词和概念引用了VWLP中所述。AI在建立或审查填写好的模板前，应该先详细阅读VWLP。如AI调用者（即创作者）未提供VWLP，则应停止阅读，向调用者反馈提供VWLP文档。
角色元设定是一个历史嵌套元设定，其由多个历史段组成。下面的模板是单个历史段的示例。创作者可以在一个角色元设定中设定多个历史段。角色元设定表的文档可以是一个节点-链接图文档，也可以是TXT等纯文字文档，这点不做限制。更多说明可参考VWLP中关于元设定表、角色元设定的叙述。

# 二、模板（单个历史段）

1、历史段所处时间区间

【注】最好和角色所处虚拟世界共用的一套时间历法。也可以用独特的编号等标识，不过最好让这个时间区间可以自然而然地融入世界元设定的历史段内。

2、角色基本信息

【注】在该历史段下。和之前历史段相同的信息可以略去。

（1）代号

【注】角色的唯一文字标识，包括但不限于角色姓名。推荐为角色姓名，如果存在特殊情况再行内部标注。

（2）年龄段

【注】角色在该历史段下的年龄段。因为存在子时空的可能性，所以历史段区间和年龄段区间不一定相等，所以这里可以再行说明。

（3）物种

【注】指生物或非生物类别，不包括所处的社会群体或所属文明。

（4）性别

【注】如有。还可做额外说明。

（5）典型外观

【注】文字描述、设定图像、模型，皆可。但是推荐在文字设定稿中使用文字或者指向非文字文件的相对路径引用。

【注】角色典型外观包括身高/体重/相貌等，这里可按需自由填写。而且可以填写多套。

（6）能力

【注】字面含义。但是也可以用TNT当量/秒之类的量化角色的破坏力。

3、角色社会信息

（1）物质资产

【注】角色所拥有的道具、财产等物质附庸。

（2）社会关系

【注】角色和其它角色在该历史段下的关系网络。这里包括两部分的内容：

1.角色和其它角色之间的关系

2.角色所处的社会群体（包括角色所处的文明、社会、部落、帮派，甚至可以细化到圈层、阶级等等）

3.角色和其它社会群体之间的关系

（3）生长环境

【注】这里包含两部分内容：

1.角色当时所身处的微观生存环境，仅涉及角色所生活的周身环境

2.角色当时所处的宏观生存环境，涉及到社会文明的局势（如果世界元设定中有提及，则可在此引用）

（4）时期遭遇

【注】角色在该历史段下、生长环境中的遭遇。

4、角色思想活动

【注】角色在该历史段下的精神活动，包括：根据其所直观看到的，所思，所学，最后是对这段时期的感悟。

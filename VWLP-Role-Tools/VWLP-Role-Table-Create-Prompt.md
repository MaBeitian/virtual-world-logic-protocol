1. 详细阅读用户提供的文件中关于《虚拟世界逻辑性协议》（即VWLP）的全文内容。如果用户未提供VWLP文档，或无法读取VWLP文档内容，则应停止工作，向调用者反馈无法阅读VWLP文档的详细原因，并提出让自己能够正确读取VWLP文档的相关建议。在编写角色元设定表时，你应该开启深度推理模式或者调用深度推理模型，进行这项工作。如果VWLP协议将占满你的上下文窗口，你应停止工作，提醒用户上下文窗口将满，并向用户说明其解决方案。

# VWLP-AI-review

按照[VWLP.md](./References/VWLP.md)中的虚拟世界逻辑性协议的规则，审查用户给出的.md文件和规则链表文件中，关于虚拟世界的设定或者剧情内容，在该协议规则下逻辑性的强弱，并将审查意见组织成.md文件，输出到用户给出的审查文件的相同目录下。

# 工作流程

1. 读取用户提供的.md文件中关于虚拟世界的设定或者剧情内容部分。如果用户给出的文档不是.md文档，那么就先确定是否能够对其进行编辑；如能编辑，则进行下一步；如果不能编辑该文档，则停止当前工作，告诉用户以下信息：“因为AI工具对.md文档普遍具有原生编辑的能力，但对doc或docx等文件的编辑能力尚需插件辅助或者不兼容。为了节省处理时间和token，请您将您的文档内容全选并复制到一个.txt文件中，之后将.txt文件的后缀改为.md后，再将该文档交给我审查就好了。本技能目前暂时只能审查.md文档和mermaid格式文档，对于图片等内容暂不支持。您可以二次开发本技能，让它具备多模态审查能力。”
2. 如果可以正常编辑用户文档，则读取用户文档中关于虚拟世界的设定或者剧情内容，按照[VWLP.md](./References/VWLP.md)中的虚拟世界逻辑性协议的规则，审查用户给出的.md文件中，关于虚拟世界的设定或者剧情内容在VLCP协议下逻辑性欠缺的地方、不符合虚拟世界逻辑性协议的规则的地方后，再整体评价用户文档逻辑性的强弱，之后将审查意见组织成一个名为“虚拟世界设定及相关内容逻辑性建议.md”的文件，并输出到用户给出的.md被审查文件的相同目录下。这一步的审查意见不需要指出用户未提供规则链表的问题。
3. 输出完毕后，向用户询问“是否制作了mermaid格式（后缀为.mmd或 .mermaid ）的规则链表”，并停下，等待用户进一步指示。若用户未提供规则链表文件，则结束工作；若用户提供了规则链表文件，则尝试读取该文件内容。倘若无法读取，则停止工作，向用户反馈无法打开该文件的具体原因。
4. 成功读取用户提供的规则链表文件后，按照[VWLP.md](./References/VWLP.md)中的虚拟世界逻辑性协议的规则和定义，审查用户给出的规则链表文件，最后将审查意见组织成一个名为“虚拟世界规则链表逻辑性建议.md”的文件，并输出到用户给出的被审查规则链表文件的相同目录下。

# 注意事项（AI使用该SKILL时，必须阅读）

1. 绝对不能修改用户提供的文件；
2. 每次使用该技能审查用户文档时，都必须查看[VWLP.md](./References/VWLP.md)，必须参考[VWLP.md](./References/VWLP.md)中的全部内容进行审查，并严格按照工作流程的要求对用户的文档内容进行审查。
3. 如果没有找到[VWLP.md](./References/VWLP.md)文件，则必须告诉使用者没有找到该文件，然后停下来等待使用者下一步的指示。
4. 输出的逻辑性建议，应使用[VWLP.md](./References/VWLP.md)中定义的术语以描述相关内容。
